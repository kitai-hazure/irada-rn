## Getting Started

Add a `.env` file in the root of the project with the following content:

```bash
WALLET_CONNECT_PROJECT_ID=
ALCHEMY_KEY=
DEV=
```

Then run the following commands:

```bash
yarn install # install dependencies
yarn start # start the app
```
